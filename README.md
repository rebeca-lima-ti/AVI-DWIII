# 🚗 AutoBots - Sistema de Gestão de Manutenção Veicular

Projeto desenvolvido como parte da atividade avaliativa de **Desenvolvimento Web III (DWIII)**.

---

## 📌 Sobre o Projeto

A **AutoBots** é uma startup fictícia criada com o objetivo de desenvolver um sistema de gestão para:

- 🔧 Lojas de manutenção veicular  
- 🛒 Vendas de autopeças  

O sistema foi projetado para ser:

- ⚡ Rápido
- 🌐 Acessível via internet
- 🔗 Integrável com outros sistemas

Para isso, foi adotada uma arquitetura baseada em **microserviços** no backend.

---

## 🎯 Objetivo da Atividade

O objetivo desta atividade é:

- Completar e/ou corrigir a implementação de um **CRUD (Create, Read, Update, Delete)**  
- Trabalhar com **entidades e regras de negócio** já parcialmente estruturadas  
- Aplicar boas práticas como:
  - 🧩 Padrões de projeto  
  - 📐 Princípios SOLID  

---

## 🛠️ Tecnologias Utilizadas

- ☕ Java  
- 🌱 Spring Framework  
- 💻 VSCode  
- 🗄️ Arquitetura baseada em microserviços  

---

## 🔄 Funcionalidades

Atualmente, o sistema contempla:

- ✅ Cadastro de clientes  
- 🔍 Consulta de clientes  
- ✏️ Atualização de dados  
- ❌ Remoção de registros
